Exercícios práticos sobre estruturas de repetição em Java, com foco em loops e iterações, fundamentais para resolver problemas que exigem execução repetida de trechos de código.

Conteúdo dos exercícios

For – loop com número conhecido de iterações

While – loop baseado em condição, executa enquanto a condição for verdadeira

Do-While – garante pelo menos uma execução antes de testar a condição

Loops aninhados – utilização de loops dentro de loops

Uso de break e continue – controle de fluxo dentro das repetições

Exemplos de exercícios:

Contar de 1 a 10 usando for

Somar números até que o usuário digite zero usando while

Criar tabelas de multiplicação com loops aninhados

Filtrar números pares em uma lista usando continue
